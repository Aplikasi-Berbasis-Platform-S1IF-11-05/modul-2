# Dasar Teori

<p align="justify">
HTML adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. HTML berfungsi untuk menampilkan berbagai elemen pada halaman web seperti teks, gambar, tabel, tautan, dan formulir sehingga dapat ditampilkan oleh browser.
</p>

<p align="justify">
HTML dikembangkan oleh Tim Berners-Lee pada awal tahun 1990-an sebagai bagian dari pengembangan World Wide Web. Bahasa ini menggunakan tag atau penanda tertentu untuk mengatur struktur dan tampilan dasar suatu halaman web.
</p>

<p align="justify">
Dalam HTML terdapat berbagai elemen penting seperti heading, paragraph, link, image, table, dan form yang digunakan untuk menyusun konten halaman. Setiap elemen ditulis menggunakan tag pembuka dan tag penutup yang mengapit isi konten.
</p>

<p align="justify">
HTML biasanya digunakan bersama teknologi web lainnya seperti CSS untuk mengatur tampilan dan JavaScript untuk menambahkan interaksi pada halaman web. Dengan kombinasi teknologi tersebut, halaman web dapat menjadi lebih menarik, dinamis, dan interaktif.
</p>
