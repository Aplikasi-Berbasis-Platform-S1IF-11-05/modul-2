## Dasar Teori
HTML adalah bahasa markup standar yang digunakan untuk membuat dan menyusun struktur halaman web. HTML berfungsi untuk menentukan berbagai elemen pada sebuah halaman, seperti teks, gambar, tautan, tabel, dan formulir sehingga dapat ditampilkan oleh browser.

HTML menggunakan tag atau penanda tertentu untuk mengatur struktur serta tampilan dasar dari konten web. Bahasa ini pertama kali dikembangkan oleh Tim Berners-Lee pada awal perkembangan World Wide Web, dan hingga sekarang HTML menjadi dasar utama dalam pembuatan website yang biasanya dikombinasikan dengan teknologi lain seperti CSS dan JavaScript.

HTML pertama kali dikembangkan pada tahun 1991 oleh Tim Berners-Lee, seorang ilmuwan komputer yang bekerja di CERN. HTML dibuat sebagai bahasa markup yang digunakan untuk menyusun dan menampilkan dokumen pada sistem World Wide Web. Pada tahap awal, HTML hanya memiliki beberapa tag sederhana yang berfungsi untuk menampilkan teks dan membuat tautan antar dokumen.

Seiring dengan perkembangan teknologi internet, HTML terus mengalami pembaruan melalui berbagai versi yang menambahkan fitur-fitur baru agar halaman web menjadi lebih interaktif dan terstruktur. Hingga saat ini, HTML berkembang menjadi standar utama dalam pembuatan website yang digunakan di seluruh dunia.
