# Dasar Teori

<p align="justify">
HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML digunakan untuk menentukan struktur dasar dari sebuah halaman web seperti judul, paragraf, gambar, tabel, tautan (link), dan berbagai elemen lainnya yang akan ditampilkan pada browser.
</p>

<p align="justify">
HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991 sebagai bagian dari pengembangan World Wide Web. Tujuan utama HTML adalah memungkinkan pengguna untuk menampilkan dan menghubungkan dokumen melalui internet menggunakan hyperlink.
</p>

<p align="justify">
HTML bekerja menggunakan tag atau penanda yang ditulis di dalam tanda kurung sudut (< >). Tag tersebut digunakan untuk memberi instruksi kepada browser tentang bagaimana sebuah konten harus ditampilkan. Sebagian besar tag HTML memiliki pasangan, yaitu tag pembuka dan tag penutup.
</p>
